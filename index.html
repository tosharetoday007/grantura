Welcome to Grantura

Set up your AI provider and install the browser extension. This takes about 2 minutes and you only do it once.

Step 1 — Choose AI Provider & Save API Key
Grantura uses free AI to read your CV and answer form questions. Choose a provider and get your free key.
Free at console.groq.com — Sign up → API Keys → Create Key. Gives 14,400 requests/day free.
Step 2 — Install Browser Extension
The extension works on Firefox, Edge, Chrome, and Brave. It activates automatically on any form page — you never need to click anything to start it.
✓
Firefox is the easiest install — no store submission needed, no special mode required. Just download and install directly.
1
Download the extension fileClick the button below — this downloads a .xpi file which Firefox recognises as an extension
2
Firefox asks to install itA popup appears: "Add Grantura to Firefox?" — click Add. That is it. Done.
3
Save your API key in the extensionClick the Grantura 🎓 icon in your toolbar → Settings tab → paste your API key → Save Key
🦊 Download for Firefox (.xpi)
ℹ
If Firefox shows a security warning, click Allow or go to about:config and set xpinstall.signatures.required to false. This is safe — Grantura is open source and never sends your data anywhere.
ℹ
Chrome requires a small one-time setup. Follow these steps carefully — it takes about 2 minutes.
1
Download and extract the zipClick the button below → save the zip → right-click it → Extract All → remember the folder location
2
Open Chrome ExtensionsIn Chrome address bar type chrome://extensions and press Enter
3
Turn on Developer ModeLook for the Developer mode toggle in the top-right corner of the page — turn it ON. This is a standard Chrome feature that allows installing extensions from files.
4
Load the extensionClick Load unpacked (appears after Developer mode is on) → navigate to the extracted folder → select the grantura-extension-v3 folder → click Select Folder
5
Save your API key in the extensionClick the Grantura 🎓 icon in your toolbar → Settings tab → paste your API key → Save Key

Fill Your Form

The extension is ready. Go to your grant portal or any website with a form — the Grantura sidebar appears automatically.

How it works on every form page
Repeat this for each page of a multi-page form.
1
Go to the formNavigate to your grant portal, Google Form, university application — any website. Log in if required.
2
Grantura sidebar appears automaticallyOn the right side of the page you will see the Grantura panel. It reads all questions on that page and calls AI to prepare answers from your documents.
3
Click "Fill This Page Now"All fields fill instantly. Fields shown in orange need your manual input — the sidebar tells you exactly which ones.
4
Come back here to reviewReturn to Grantura → Review & Save to see what was filled, edit anything needed, and save a printable copy.
5
Click Next in the formGo to the next page of the form — the sidebar appears again and fills that page too. Repeat until the form is complete.
